# goit-algo-hw-07

## Реалізація

- `tree_utils.py` містить реалізацію двійкового дерева пошуку з операціями вставки та утилітами:
  - `find_maximum(node)` — знаходить найбільше значення (Завдання 1).
  - `find_minimum(node)` — знаходить найменше значення (Завдання 2).
  - `sum_values(node)` — повертає суму всіх значень у дереві (Завдання 3).
  - `build_sample_tree()` створює приклад дерева для демонстрації.
- `task-1.py`, `task-2.py`, `task-3.py` викликають відповідні функції та друкують результати.

## Запуск прикладів

```bash
cd goit-algo-hw-07
python3 task-1.py   # максимальне значення
python3 task-2.py   # мінімальне значення
python3 task-3.py   # сума всіх значень
```
